## About

This repository contains projects on higher level
programming using `Python` and `JavaScript` programming languages.
The concepts covered:

* Classes and objects
* Object-oriented programming principles:
  * Inheritance
  * Encapsulation
  * Abstraction
  * Polymorphism
* SQL queries
* SQL joins
* Unit testing
* Object scopes
* Exceptions
* Imports and modules
